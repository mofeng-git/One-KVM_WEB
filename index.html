    介绍

    项目简介

    One-KVM 是基于廉价计算机硬件和 PiKVM 软件二次开发的 BIOS 级远程控制项目。可以实现远程管理服务器或工作站，无需在被控机安装软件调整设置，实现无侵入式控制，适用范围广泛。

    • 安装迅速

      使用 Docker 或整合包，即刻完成部署，开箱即用

    • 非侵入式控制

      无需在被控机安装软件，实现非侵入式控制

    • 功能丰富

      KVM 远程控制、ATX 电源管理、 USB 虚拟外设、VNC、H.264/WebRTC 等

    • 开源开放

      基于 PiKVM，以 GPLv3 开源许可证在 GitHub 开源
